First Program

Every summer, my brother and I went to my grandmother's country house. One evening my brother had an interesting idea. There are closed channels/chats in the telegram and links to these channels looked like this: https://t.me/joinchat/AAAAA {17 random characters of the upper and lower case English alphabet + digits} we also noticed that when these links opened through the browser, the working link opens several times slower than the non-working one. On this basis, we decided to make a bot in the python programming language, which will randomly iterate (bruteforce) links to closed channels.

It was my first python program that my brother and I wrote on the phone. After a few days, the search became a little more difficult due to the addition of -_ to random characters, but in any case, we added multithreading and ran this bot for several days to find something. The results are obviously fail because of 64^17 variations of link XD

Attempts

From that moment on, my attempts to find a niche suitable for me in programming began. After several years of development, not surprisingly, my choice fell on python, simply because there was more practical use in it. There was a time when I did all kinds of bots/python programs all day long, because I had to solve some kind of one-time problems, either quickly or in the short term. Python is the best choice for such tasks because of the high speed of writing code and its compactness. There were many attempts to leave for GameDev, but at the last one I concluded for myself that it was not mine and decided that I would return there only if I worked, or if I was called to the project.

I haven't been particularly involved in creating applications for android, but I think it can be quite suitable for itself. Creating websites, always shunned this topic, there was no desire. There were also many attempts to get into BigData and AI, but the lack of knowledge from higher mathematics put a barrier. And so it turned out that software development has become my hobby for several years
